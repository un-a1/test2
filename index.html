HTML Living Standardとは

HTML Living Standardとは、Apple・Mozilla・Operaの開発者によって設立された「WHATWG」という組織が策定するHTMLの標準仕様で、 HTML5の廃止によって標準仕様となり、HTML Living Standardで適切なマークアップをすれば、正しいHTMLとしての品質が保たれることになる。

HTML Living Standard

W3C（ The World Wide Web Consortium ）

HTMLとCSSの仕様について理解しておく必要があります。これらの技術の大半は、 特定の企業が開発した独自の技術ではなく、 「W3C（The World Wide Web Consortium）」という ウェブ技術の標準化を推進する組織によって策定されていました。

WHATWG （ Web Hypertext Application Technology Working Group ）

ウェブ技術の標準化の推進を、「W3C（The World Wide Web Consortium）」によって策定されていたが、現実的な運営方針に徐々に合わなくなり、 GoogleのChromeやMozillaのFirefox,、AppleのSafari,、Operaなどの主要ブラウザがW3Cに準拠することを辞め、 「WHATWG（Web Hypertext Application Technology Working Group）」を独自運営しはじめる。

HTML Living Standardに対応ラウザ

主要なWebブラウザは、HTML 5.1の勧告があった2016年ごろから、HTMLの標準仕様をHTMTL5からW3Cに準拠することを辞めて、 HTML Living Standardを標準仕様として採用。
また、唯一MicrosoftのEdgeだけはHTMTL5に準拠していたが、2018年12月にChromium ベースに移行することを決定し、 2021年現在、主要なWebブラウザはすべてHTML Living standardに準拠している。
